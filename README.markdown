Rushed Dinosaur Dare
====================

Rushed Dinosaur Dare is a game I made for the Ludum Dare 36 48h Compo.

Build from Source
-----------------

* Install a2x from [here](https://github.com/alxm/a2x)
* Clone this repo, `cd make && make` or `make -f Makefile.mingw`
